# Wireless ADB Connector

- Devices ADB Connector via Wi-Fi

## Features
- Connect your device via ADB to your computer via Wi-Fi for the purpose of debugging projects in Flutter and perhaps other cross-platform projects (not tested) for Android.

## How to use
- Install Android SDK (Only the first time)
- Set the path of adb into System Variables (Only the first time)
- On your device check which WiFi IP (Only the first time or when your device's IP changes)
- Connect the cable to your phone
- Run the project
- Disconnect the USB cable
